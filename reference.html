Code Stack

This site was created using JavaScript, HTML, and CSS. HTML was used for the main content and structure of the website, CSS was used to design the majority of the website, and JavaScript was used to implement additional functionality features for users. We chose these languages because they are widely supported and allow for efficient development and iteration.

Image Links

Almost all images used in this website were provided by the Jobees Robotics team themselves, with the exception of the FIRST Robotics information page. The logo was created using Canva and Adobe Illustrator, and all event photos were created by the organization using a variety of design softwares.
